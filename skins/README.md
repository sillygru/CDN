Skin model showcases for brawlers, linked by their IDs from [skins](https://api.brawlify.com/game/csv_logic/skins).

# Standards

## model
Full-body front-facing model showcase of each skin. Dimensions vary by character proportions and skin size, up to ~1100px on the longest side. All assets must be compressed.
